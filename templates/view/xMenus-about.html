xMenus Bootstrap Menu

Bootstrap Menu is flexible and responsive components.It is a Client Side Components.It is simple, effective and easy to use.

Major features of Bootstrap Menu :

  • User can add multiple menus in single site.
  • Sortable menu items.
  • It is completly responsive and dynamic Bootstrap Menu
  • User can change the basic properties like padding, margin, background, border etc. through component tool box.
  • User can add Multiple Pages but Menu Caption Must be required.
  • User can add Sub menu items in parent menu.
